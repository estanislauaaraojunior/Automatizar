# Bots e Automações de Redes Sociais

Esta pasta contém bots e automações para diferentes plataformas.

## Exemplos de Bots

- Bots do Telegram
- Bots do Discord

## Observações Importantes

- Sempre respeite os termos de serviço das plataformas
- Use rate limiting para evitar bloqueios
- Mantenha tokens e credenciais em variáveis de ambiente

# Automação Web

Esta pasta contém projetos de automação web usando ferramentas como Selenium, Puppeteer, Playwright, etc.

## Ferramentas Comuns

- Selenium (Python/Java/JavaScript)
- Puppeteer (Node.js)
- Playwright (Node.js/Python)
- BeautifulSoup (Web Scraping)

## Casos de Uso

- Web scraping
- Testes automatizados
- Preenchimento de formulários
- Coleta de dados
- Automação de tarefas repetitivas

## Boas Práticas

- Use headless mode quando possível
- Implemente delays adequados
- Trate exceções corretamente
- Respeite robots.txt

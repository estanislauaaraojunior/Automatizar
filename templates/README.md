# Templates

Templates e estruturas básicas para novos projetos de automação.

## Templates Disponíveis

Cada template fornece uma estrutura inicial para começar rapidamente um novo projeto de automação.

## Como Usar

1. Copie o template desejado
2. Renomeie conforme seu projeto
3. Customize conforme necessário
4. Remova arquivos de exemplo
5. Atualize o README

## Estrutura de Template

Cada template deve incluir:
- README.md com instruções
- .env.example para variáveis de ambiente
- Arquivo de dependências (requirements.txt, package.json, etc.)
- Estrutura básica de pastas
- Exemplos de código comentados

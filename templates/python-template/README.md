# Template Python - Projeto de Automação

Template básico para criar projetos de automação em Python.

## Estrutura do Projeto

```
python-template/
├── README.md           # Este arquivo
├── requirements.txt    # Dependências Python
├── .env.example       # Exemplo de variáveis de ambiente
├── main.py            # Script principal
└── utils.py           # Funções auxiliares
```

## Pré-requisitos

- Python 3.8 ou superior
- pip (gerenciador de pacotes Python)

## Instalação

1. Copie este template para um novo diretório
2. Renomeie conforme seu projeto
3. Instale as dependências:

```bash
pip install -r requirements.txt
```

4. Configure as variáveis de ambiente:

```bash
cp .env.example .env
# Edite o arquivo .env com suas configurações
```

## Uso

```bash
python main.py
```

## Customização

1. Edite `main.py` com sua lógica de automação
2. Adicione funções auxiliares em `utils.py`
3. Atualize `requirements.txt` com novas dependências
4. Documente as mudanças neste README

## Dependências

Veja `requirements.txt` para a lista completa de dependências.
